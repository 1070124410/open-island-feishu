## 修复

- **收起态 agent 计数**：右侧 `×N` 与顶部「运行」统计现在会计入 Claude Code 子代理（subagent），不再只统计 session 数
- **代理网格模式**：每个运行中的 subagent 展开为独立色块，不再把 5 个子代理折叠成 1 个

## 从 v0.0.3 升级

应用内 **设置 → 关于 → 检查更新**，或手动下载 DMG 安装。

## 系统要求

- macOS 14.0+
- 飞书 sidecar：[vibe-island-feishu](https://github.com/1070124410/vibe-island-feishu)
