# Open Island Feishu v0.0.3

## 下载

| 文件 | 说明 |
|------|------|
| `Open Island Feishu.dmg` | 推荐 — **Universal（Intel + Apple Silicon）** |
| `Open Island Feishu.zip` | Sparkle 应用内更新使用此 zip |

## 新功能

- **应用内自动更新**：基于 GitHub Releases + Sparkle，设置 → 关于 → 检查更新
- **Universal 二进制**：同时支持 Intel Mac 与 M 系列芯片
- 延续 v0.0.2：自定义文本滚动方向可配置

## 从 v0.0.2 升级

v0.0.2 无应用内更新源，请**手动安装一次** v0.0.3；之后可在应用内检查更新。

## 系统要求

- macOS 14.0+
- 飞书 sidecar：[vibe-island-feishu](https://github.com/1070124410/vibe-island-feishu)
