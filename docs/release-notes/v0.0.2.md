# Open Island Feishu v0.0.2

## 下载

| 文件 | 说明 |
|------|------|
| `Open Island Feishu.dmg` | 推荐 — 拖入 `/Applications` |
| `Open Island Feishu.zip` | 备用 zip |

## 新功能

- **自定义文本滚动方向**：开启「滚动播放」后，可选择 **从左到右** 或 **从右到左**（设置 → 个性化 → 自定义 → 文本）

## 系统要求

- macOS 14.0+
- 飞书远程审批需安装 [vibe-island-feishu](https://github.com/1070124410/vibe-island-feishu) sidecar
