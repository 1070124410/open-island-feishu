## 新增 / Changes

### 多显示器 placement 体验改进

- **外接屏 panel 紧凑化**：在没有刘海的外接屏上，Island panel 改用 480×96 的紧凑尺寸，不再以 ~680×150 的"大窗口"漂浮在屏顶。刘海屏行为不变。
- **显示器选择反馈可见**：`设置 → 显示` 中，
  - Picker 选项加副标题：例 `Built-in Retina Display — Built-in notch · 3024×1964` / `DELL XXX — Top-bar fallback · 1920×1080`，让你切换前知道目标屏属于哪一类
  - Diagnostics 区域新增高对比度 callout（SF Symbol + 强调色），切显示器后**实时**显示当前 placement mode（"Notch area" 蓝色 / "Top bar fallback" 橙色）

### Multi-display placement UX (EN)

- **Compact external-display panel**: on screens without a notch, the Island panel now uses a 480×96 compact size instead of floating as a ~680×150 window. Notch displays are unchanged.
- **Visible display-mode feedback**: under `Settings → Display`,
  - Picker options now include subtitles such as `Built-in Retina Display — Built-in notch · 3024×1964` / `DELL XXX — Top-bar fallback · 1920×1080`, so you can tell what kind of screen each option represents.
  - Diagnostics section gains a high-contrast callout (SF Symbol + accent tint) that updates **live** to show the active placement mode ("Notch area" in blue / "Top bar fallback" in orange) whenever you switch displays.

## 已知问题 / Known limitations

- **External-display closed bar 视觉粗糙**：在 480×96 紧凑 panel 下，`V6ClosedPill` 的 external layout 没有专门调过，比 v0.0.4 看起来更不打磨。下一版会重做。
- **External-display opened state 内容会被裁剪**：480×96 panel 容不下 session list / approval card / question card 的完整内容。如果你常用外接屏接收审批，建议保持 picker 选 `Automatic`（仍走刘海屏）。下一版考虑改 detached popup 或 floating menubar 方案。
- **External-display closed bar may look rough**: the external `V6ClosedPill` layout was not retouched for the new 480×96 panel and is less polished than v0.0.4. To be rebuilt in the next release.
- **Opened state may be cropped on external displays**: approval / question / completion cards do not fit in the 480×96 panel. Keep the display picker on `Automatic` if you rely on the panel for approval workflows. A detached popup window is on the roadmap.
