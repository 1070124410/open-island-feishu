# Open Island Feishu v0.0.1

首个公开发布版本。

## 下载

| 文件 | 说明 |
|------|------|
| `Open Island Feishu.dmg` | 推荐 — 拖入 `/Applications` 安装 |
| `Open Island Feishu.zip` | 备用 zip 包 |

## 新特性

- 基于 [Open Island v1.1.3](https://github.com/Octane0411/open-vibe-island) 的飞书远程审批集成
- 独立应用名 **Open Island Feishu**（可与官方 Open Island 并存）
- Bundle ID：`app.openisland.feishu`
- 设置 → 飞书远程审批（新手引导、Hook 接入、测试卡片）
- 灵动岛自定义左槽（Scout / 文本 / 图片）、文本滚动与展示宽度
- 禁用上游 Sparkle 自动更新，避免覆盖飞书集成

## 系统要求

- macOS 14.0+
- 飞书 sidecar：[vibe-island-feishu](https://github.com/1070124410/vibe-island-feishu)

## 安装 sidecar

```bash
git clone https://github.com/1070124410/vibe-island-feishu.git ~/open-island-feishu
cd ~/open-island-feishu && ./scripts/install.sh
```
